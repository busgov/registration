Create an account help

To create an account follow the steps below:

  1. Enter your email address below
  2. Select the Send verification code button.
  3. Check your email account for an email arriving shortly from "Microsoft on behalf of business.gov.au".
  4. Copy the code from your email and return to this page.
  5. Enter the verification code into the box that'll appear below.
  6. Select the Verify code button.
  7. If this doesn’t work, select the Send new code button and repeat steps 3 to 6.
  8. Enter a password 8 - 16 characters long with at least 3 of the following:
    • lowercase letters
    • uppercase letters
    • numbers
    • one or more of the following symbols:
      @ # $ % ^ & * - _ + = [ ] { } | \ : ‘ , ? / ' ~ " ( ) ; .
  9. Fill in the rest of the details and select the Create button.

Need more help? Contact us via business.gov.au or call 13 28 46.
