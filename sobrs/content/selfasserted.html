How to create a new account

To create an account follow the steps below:

  1. Enter your email address below
  2. Select the Send verification code button (this emails you the code you need to continue)
  3. Find the verification code email from “Microsoft on behalf of business.gov.au”.
  4. Copy the code from your email and return to this page.
  5. Pop the verification code in the box that’ll appear below.
  6. Select the <Verify code button.
  7. Create a password 8 - 16 characters long with at least 3 of the following:
    • lowercase letters
    • uppercase letters
    • numbers
    • one or more of the following symbols:
      @ # $ % ^ & * - _ + = [ ] { } | \ : ‘ , ? / ` ~ “ ( ) ; .
  8. Fill in the rest of the details and select the Create button.

Still having trouble? Contact us via business.gov.au or call 13 28 46.
